// 目录说明
smartva是智能影音产品应用群，story(故事机)是具体某一个应用，后续可以添加n个类story的应用。	
.
├── common
│			├── common.h		// 对外头文件，上层使用该层，包含该头文件
│			├── business		// 业务层，自定义业务逻辑以及与第三方库对接，为上层提供接口
│     │      ├── va_business.h		// 对外头文件
│     │      ├── cJSON.c					// cjson库
│     │      ├── cJSON.h
│     │      ├── ...
│     │      ├── page.c						// gui界面切换自定义
│     │      ├── page.h
│     │      ├── ...
│			├── driver			// 驱动对接层，为上层提供统一的接口，屏蔽平台硬件驱动的差异
│     │      ├── va_driver.h				// 对外头文件
│     │      ├── va_audio.c					// 与audio驱动对接
│     │      ├── va_audio.h
│     │      ├── ...
│     │      ├── va_display.c				// 与显示驱动对接
│     │      ├── va_display.h
│     │      ├── ...
│			├── middleware			// 中间件对接层，为上层提供统一的接口，屏蔽中间件的差异
│     │      ├── ...
│     │      ├── ...			
│     │      ├── ...
├── story
│			├── business					// 业务层，与GUI无关的业务逻辑
│     │      ├── main.c			// 主函数入口
│     │      ├── ...
│     │      ├── ...
│			├── config						// 配置层
│     │      ├── lv_conf.h					// littlevgl核心库配置裁剪
│     │      ├── lv_drv_conf.h			// littlevgl驱动对接库配置裁剪			
│     │      ├── ...
│     │      ├── ...
│			├── designer					// gui工具设计层
│     │      ├── fonts			// 字体资源
│     │      ├── images			// 图片资源
│     │      ├── lvgl 			// .lvgl可视化文件
│     │      └── src				// 代码
│     │           ├── moc_home.c		// moc业务逻辑层，与GUI有关的业务逻辑，包含部分UI代码，工具创建产生代码模板一次，之后需要手动添加
│			│						├── ui_home.c			// ui界面， 图形界面工具产生，不可以手动修改。
│			│						├── ...          
│			│						├── ...
